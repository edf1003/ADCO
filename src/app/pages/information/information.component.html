Introducción al tema

El objetivo de este proyecto es desarrollar una aplicación web que permita a los usuarios realizar un análisis completo de conjuntos de datos proporcionados por ellos mismos. La aplicación se enfoca en cuatro técnicas de análisis: Componentes Principales, Distancias, Clustering y Detección de Outliers.

En particular, la aplicación proporciona los siguientes casos de uso:

  • Obtener los datos introducidos por parte del usuario.
  • Llevar a cabo un análisis de componentes principales.
  • Llevar a cabo un análisis de distancias (Euclidea, Euclida normalizada y Mahalanobis).
  • Llevar a cabo un análisis de clustering (DBSCAN, OPTICS y K-means).
  • Detectar outliers en los datos mediante los algoritmos (Mahalanobis, DBSCAN, OPTICS, KNN y LOF).


La aplicación web se está a disposición de la comunidad científica de forma gratuita y bajo una licencia GNU GPL 3.0.


Instrucciones básicas de uso:

Para el correcto uso de la aplicación web, el usuario debera primeramente introducir los datos con los que se va a trabajar en la pestaña Empezar > Datos, esto se puede hacer de forma manual o subiendo un archivo de tipo .xlsx.
Una vez realizada la subida de los datos, el usuario podra interactuar con las pestaña ACP, Distancias, Clustering y Outliers, pudiendo realizar los diferentes análisis, correspondientes a cada ventana. Finalmente, en cada ventana el usuario podrá guardar una copia de los datos obtenidos durante la ejecución.
