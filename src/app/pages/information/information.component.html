Introducción al tema

El objetivo de este proyecto es crear una aplicación web capaz de realizar análisis de componentes principales, de clustering y de detección de outliers sobre un conjunto de datos aportados por el usuario. Para ello, la aplicación deberá poder calcular diferentes tipos de distancias entre puntos.
La implementación de esta aplicación requiere llevar a cabo varios análisis matemáticos, a partir de datos proporcionados por el usuario.

En particular, la aplicación deberá:

  • Obtener los datos introducidos por parte del usuario.
  • Llevar a cabo un análisis de componentes principales.
  • Incluir gráficas para los 2 primeros componentes principales.
  • Llevar a cabo un análisis de clustering utilizando diferentes algoritmos (DBSCAN, OPTICS, etc.), basados en diferentes tipos de distancias (euclídea, euclídea normalizada, Mahalanobis, etc.).
  • Detectar outliers en los datos, usando diferentes tipos de distancias (euclídea, euclídea normalizada, Mahalanobis, etc.).


La aplicación web se pondrá a disposición de la comunidad científica de forma gratuita. La aplicación debe ser web, gratuita, y cederse bajo licencia GNU GPL 3.0. Se realizará sobre Angular (HTML, CSS y JS).

Instrucciones de uso:

Para el correcto uso de la aplicación web, el usuario debera primeramente introducir los datos con los que se va a trabajar en la pestaña Empezar > Datos, esto se puede hacer de forma manual o subiendo un archivo de tipo .xlsx.
Una vez realizada la subida de los datos, el usuario podra interactuar con las pestaña ACP y Clustering pudiendo realizar los diferentes anaálisis, correspondientes a cada ventana. Finalmente, en la ventana de Resultados el usuario podrá guardar una copia de los datos obtenidos durante la ejecución.
