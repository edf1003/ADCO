TFG - GII 22.40 Aplicación web para realizar análisis de componentes principales, de clustering y de detección de outliers.

Información Datos 3 primeros componentes Resultados Outliers

Introducción al tema:

El objetivo de este proyecto es crear una aplicación web capaz de realizar análisis de componentes principales, de clustering y de detección de outliers sobre un conjunto de datos aportados por el usuario. Para ello, la aplicación deberá poder calcular diferentes tipos de distancias entre puntos, incluyendo la distancia de Mahalanobis. La implementación de esta aplicación requiere llevar a cabo varios análisis matemáticos, a partir de datos proporcionados por el usuario.

En particular, la aplicación deberá:

      - Graficar los datos de diferentes formas.
      - Llevar a cabo un análisis de componentes principales.
      - Incluir gráficas para los 3 primeros componentes principales.
      - Llevar a cabo un análisis de clustering utilizando diferentes algoritmos (DBSCAN, OPTICS, etc.), basados en diferentes tipos de distancias (euclídea, euclídea normalizada, Mahalanobis, etc.).
      - Graficar los resultados de diferentes formas.
      - Detectar outliers en los datos, usando diferentes tipos de distancias (euclídea, euclídea normalizada, Mahalanobis, etc.).


La aplicación web se pondrá a disposición de la comunidad científica de forma gratuita. La aplicación debe ser web, gratuita, y cederse bajo licencia GNU GPL 3.0. Se realizará sobre Angular (HTML, CSS y JS).
